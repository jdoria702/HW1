Credit Card Fraud Detection

Machine learning pipeline for detecting fraudulent transactions using data preprocessing techniques, imbalanced classification techniques and evaluation metrics

Python | scikit-learn | pandas | Seaborn | imbalanced-learn | Linux
  • Preprocessed 240,000+ Kaggle credit card transactions using UNIX tools (grep, cut, uniq, sort) to remove duplicates and missing values in the original dataset
  • Addressed class imbalance via oversampling and undersampling with scikit-learn
  • Plotted results of resampled dataset along with dataset dimensions with Seaborn to visualize samples added or removed as a result of resampling techniques
  • Trained and evaluated 3 ML models (logistic regression, random forest classifier, multi-layered perceptron) using K-Fold Cross Validation and hyperparameter tuning to find model and parameters with best results
